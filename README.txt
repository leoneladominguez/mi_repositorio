# LD-Desing Gozo-WEB

Sitio web de GOZO.

## Acerca de

Este proyecto es un sitio web construido en HTML y  CSS redactado en SASS para dar soporte a la version WEB de GOZO.

## Contenido

- **index.html**: Este archivo es la página principal del sitio web.
- **style.css**: Archivo de hojas de estilo CSS que define el diseño y la apariencia del sitio.

## Uso

Para ver el sitio web, abrir el archivo `index.html` en un navegador web.
